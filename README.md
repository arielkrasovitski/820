## My data project for CIND 820 

## Overview 
This project analyzes a self-reported dataset consisting of student awareness of social media addiction, using consistent metrics and changing focus factors to identify which elements impact the results. 

## Project Stages
1. Data Collection: Collected from www.kaggle.com/datasets/adilshamim8/social-media-addiction-vs-relationships/data
2. Data Cleaning: Handled missing values, though none were present in the dataset 
3. Data Transformation: Converted categorical variables into a numerical format in order to process them for machine learning
4. Modeling: Built and evaluated three regression models
